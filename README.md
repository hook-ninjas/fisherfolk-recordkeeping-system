# **Usage**

after cloning or forking the repo on the root folder run:

```
npm i
```

## How to Start:

To run in Development mode on the root folder run:

```js
npm run dev
```

To run a specific package or app:

```js
cd <package\app>
```

then

```js
npm start
```

or

```js
npm run dev
```

## Installing packages:

go to the directory where you want to install package then:

```js
npm i <package>
```
